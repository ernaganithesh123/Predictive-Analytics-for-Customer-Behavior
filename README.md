# 📊 Predictive Analytics for Customer Behavior

## 📌 Project Overview

This project focuses on analyzing customer behavior and predicting customer churn using data analytics and machine learning techniques. The project follows an end-to-end data science workflow, including data cleaning, exploratory data analysis (EDA), Power BI dashboard development, feature engineering, and predictive modeling.

The objective is to identify the key factors influencing customer churn and generate actionable business insights that can help organizations improve customer retention strategies.

---

## 🎯 Project Objectives

* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA).
* Generate business insights using statistical analysis.
* Develop an interactive Power BI dashboard.
* Engineer new features for improved model performance.
* Build and evaluate machine learning models for churn prediction.

---

## 📂 Dataset

The project uses a customer churn dataset containing demographic information, subscription details, customer behavior, and spending patterns.

### Dataset Files

* **Original Dataset** – Raw customer churn dataset.
* **Cleaned Dataset** – Dataset after preprocessing and data cleaning.

## 🔍 Key Business Insights

* Customers with frequent support calls are more likely to churn.
* Monthly contract customers have the highest churn rate.
* Higher payment delays are associated with increased churn.
* Customers with lower total spending are more likely to leave the service.
* Age and customer interaction history significantly influence churn.
